## 準備
   まずVS codeを開き、ファイル/新しいファイル...  を選択してください。  
   入力を求められるので「practice.ipynb」と入力して、ファイルを作成・保存してください。  
   ※practice.ipynbは、いわゆるJupyter notebookと呼ばれるもので、使い方は検索すると色々でてきます。  
   
   簡単に説明すると、「**+コード**」を押すと、”**セル**”と呼ばれるプログラムを書けるエリアが作られます。  
   そこに、pythonコードを打ち込み、セルの左端にある「**▶️ボタン**」を押すとコードを実行できます。  
   実行後はまた「+コード」を押して、新しいセルに書き込んでいく、という流れです。

   ※初回の実行時に、「Python環境の選択」みたいなことを聞かれるかもです。  
   この場合は、「Python環境...」→ "おすすめ"と書かれたものを選べば良いと思います。
